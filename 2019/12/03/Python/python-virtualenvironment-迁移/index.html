python virtualenvironment 迁移

python virtualenvironment的便利之处就在于构建了独立的python运行环境，工程所需要的环境可以很方便地打包随程序迁移到其他路径/其他机器上。

最开始以为python虚拟环境直接跟随工程打包起来，到其他机器就可以用了，真的是too young too naive…然后就收到报错啦。
